# 🌿 FinAesthetic - Smart Finance Tracker

**FinAesthetic** is a minimalist, elegant, and user-friendly web application designed to help you manage your personal finances with ease. Track your income, monitor expenses, and stay within your monthly budget—all in one aesthetic dashboard.

## 🔗 Repository Link
Official Repository: [https://github.com/SewminiDilshani/FinAesthetic-App.git](https://github.com/SewminiDilshani/FinAesthetic-App.git)

## ✨ Key Features
* **📊 Dynamic Spending Analysis:** Visualize your expenses with an interactive doughnut chart.
* **💰 Budget Management:** Set a monthly spending limit with a real-time progress bar.
* **📄 PDF Report Generation:** Export your entire transaction history into a professional PDF.
* **🌓 Dark & Light Mode:** Seamlessly switch between themes.
* **💡 Smart Insights:** Automatically identifies your highest spending category.
* **🔍 Search & Filter:** Quickly find past transactions.
* **🎵 Interactive Feedback:** UI sounds for adding or deleting records.

## 🛠️ Built With
* **HTML5 & CSS3** - Responsive Design & Modern UI.
* **JavaScript (ES6)** - Core logic & LocalStorage.
* **Chart.js** - For beautiful data visualization.
* **jsPDF & autoTable** - For generating PDF reports.

## 📂 How to Use
1. **Add Transaction:** Enter description, amount, and category.
2. **Set Budget:** Click "Set Limit" to define your monthly goal.
3. **Monitor:** Check the chart to see your spending habits.
4. **Download:** Click "Download PDF Report" to save your records.
